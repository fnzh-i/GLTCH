# GLTCH (Creative Campaign & Multi-Step E-Commerce)
GLTCH is a high-octane digital experience where the boundaries between design and code dissolve. This platform serves as the digital gateway for a 3-day tech-and-art conference, featuring a dynamic event scheduler and a foolproof, multi-step ticket configurator designed to eliminate friction and maximize conversion.

🔗 Repository: https://github.com/fnzh-i/GLTCH

## 🚀 Tech Stack
**Frontend:** React / Next.js (App Router)

**Animations:** Framer Motion (High-energy CSS transitions & Glitch effects)

**Styling:** Tailwind CSS (Brutalist Dark-Mode UI)

**Icons/Media:** Lucide-React & Optimized WebP/MP4 assets

---

## 🛠️ Local Development (Quick Start)
Since this is currently a frontend-focused build, you only need Node.js to get the experience running.

1. **Clone and Enter:**
   ```bash
   git clone [https://github.com/fnzh-i/GLTCH-Festival-Campaign](https://github.com/fnzh-i/GLTCH-Festival-Campaign)
   cd GLTCH-Festival-Campaign
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Launch the Experience:**
   ```bash
   npm run dev
   ```
   *The campaign site is now live at `http://localhost:3000`*

## 🎨 Design Philosophy
GLTCH utilizes a **Brutalist Dark-Mode** aesthetic. Key UI elements include:
- **Kinetic Typography:** Text that reacts to scroll and hover.
- **Magnetic Buttons:** Interactive elements that "pull" the cursor in.
- **Non-Linear Navigation:** A guided flow that feels like an app rather than a traditional website.
